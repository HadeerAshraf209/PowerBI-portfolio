# Data Professions Survey Analysis Dashboard (Guided Project)

## Project Overview
This is a guided Power BI project created by following
**"Full Power BI Guided Project"**
by Alex The Analyst.

The goal of this project was to build a complete analytical dashboard and practice working with survey data using Power BI.

## Objectives
- Analyze demographics of data professionals
- Understand salary trends across roles, industries, and countries
- Explore job satisfaction and work-life balance
- Identify preferred programming languages by job title

## Data Source
- Data professions survey dataset

## Key Metrics & Insights
- **Total survey participants:** 630  
- **Average age:** 29.87 years  
- **Salary satisfaction score:** 4.27 / 10  
- **Work-life balance score:** 5.74 / 10  
- Data Scientists reported the highest average salary among roles  
- Python is the most preferred programming language across job titles  

## Tools Used
- Microsoft Power BI
- Power Query
- Data modeling
