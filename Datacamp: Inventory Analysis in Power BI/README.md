## Project Overview
This project presents an end-to-end inventory analysis built in Power BI, based on a real-world business scenario from a guided DataCamp case study.
The objective is to support inventory optimization and purchasing decisions by transforming raw sales and inventory data into actionable management insights through data cleaning, DAX calculations, and interactive dashboards.
The analysis is conducted for a fictitious retail company, **WarmeHands Inc.**, facing common inventory management challenges such as inefficient stock allocation and unclear product performance.

---

## Business Objectives
- Improve visibility into inventory performance  
- Identify high- and low-value products  
- Evaluate inventory efficiency and turnover  
- Support smarter purchasing and stock control decisions  

---

## Tools & Technologies
- Power BI  
- Power Query (data cleaning & transformation)  
- DAX (business and inventory metrics)  
- Data Modeling  
- Data Visualization & Dashboard Design  

---

## Analysis Workflow

### 1. Exploratory Analysis & Data Preparation
**Goal:** Ensure the dataset is clean, reliable, and ready for analysis.

**Key Activities:**
- Data preparation and transformation using Power Query  
- Cleaning inconsistent and missing values  
- Validating business logic within sales and inventory data  
- Creating core financial metrics:
  - Cost of Goods Sold (COGS)
  - Revenue
  - Profit  

**Outcome:**  
A validated and structured dataset suitable for inventory performance analysis.
